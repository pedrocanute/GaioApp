# GaioApp

Aplicativo desenvolvido em SwiftUI para simular o cuidado com um girassol em diferentes condições de cultivo.

## Sobre o projeto

O projeto permite que o usuário interaja com variáveis como água, sol e fungicida, observando como essas condições afetam o estado visual da planta. A proposta é criar uma experiência simples, educativa e visual sobre causa e efeito no cultivo de um girassol.

## Funcionalidades

- Simulação de ambiente interno e externo
- Controle de variáveis por sliders
- Mudança visual da planta conforme os valores escolhidos
- Interface adaptada para diferentes tamanhos de tela
- Navegação entre telas usando SwiftUI

## Tecnologias utilizadas

- Swift
- SwiftUI
- Xcode
- MVVM
- Assets personalizados

## Estrutura do projeto

O projeto está organizado em pastas como:

- `Model`
- `View`
- `ViewModel`
- `Router`
- `Assets.xcassets`

Essa estrutura separa a interface, os dados, a lógica de estado e a navegação do app.
